前端界面使用bootstrap,后端使用thinkphp。前台具备信息提交和验证功能。后台具备基本的登录、退出、分页；数据遍历；表单数据验证；增删改查等功能。算是对php和mysql相关知识的一次实践。（前台界面和逻辑在Home文件夹；后台界面和逻辑在Admin文件夹；css、js文件还有图片在Public文件夹。）
